In honor of international π-day, here is a three hundred and fourteen byte JavaScript snippet Adam and I wrote a few years back that should reenable the functionality of the blink tag on any page:

{% highlight javascript %} (function(){ b = document.getElementsByTagName('blink'); for(i=0;i < b.length;i++) b[i].style.textDecoration = 'none'; function ch(d1, t1, d2, t2) { for(var i=0;i < b.length;i++) b[i].style.visibility = d1; setTimeout(function(){ch(d2, t2, d1, t1)}, t1); } ch('visible', 750, 'hidden', 250); })() {% endhighlight %}

The one cool thing about this code is that recreates the exact blinking interval that I remember from Netscape. Here's a demo page, though this should be blinking for you right now.
